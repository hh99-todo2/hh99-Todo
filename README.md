# TodoList (Lv2 과제)
****
### 기능구현
* UI 구현
* Todo 추가
    * 제목과 내용에 글자 수 제한
* Todo 취소/완료
* Todo 삭제
* Todo 상세로 보기
* 상세에서 이전으로 넘어가기
* 미입력시 alert

### 파일구조
* components
    * Todo.jsx(삭제, 취소/완료)
* pages
    * Todo.css
    * Todos.js(메인 페이지)
    * TodoDetail.js(todo상세페이지)
* redux
    * config
        * configStore.js
    * modules
        * todos.js 
* shared
    * Router.js
* App.js
* index.js
